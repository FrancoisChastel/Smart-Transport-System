Francais:
Afin de lancer la démo veuillez suivre la démarche suivante:

1. Créer une bdd en local avec MySql serveur par exemple
2. Nommer la BDD: ‘sti’
3. Créer un utilisateur avec login: ‘root’ et mdp: ‘’ (vide).
4. Importer le script baseDeDonnee.sql dans la base de donnée
5. Créer un projet sous votre IDE
6. Importer le développement situé: Dev/SystemeTransportInteligent_Communication/*
7. Ajouter dans votre classPath les deux fichiers com.mysql.jdbc_5.1.5.jar et ojdbc7.jar présents dans Dev/librairies_externes
8. Lancez la classe Main

Respectez bien toutes les étapes.
